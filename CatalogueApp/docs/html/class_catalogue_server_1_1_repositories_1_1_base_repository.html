Detailed Description

Generic base repository providing common CRUD (Create, Read, Update, Delete) operations for entities.

Template Parameters
TThe type of entity this repository manages. Must have a parameterless constructor.

This base repository implements common database operations using SQLite. All entity-specific repositories should inherit from this class to maintain consistency in basic CRUD operations across the application.
